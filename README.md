## Features

- Built using the latest **Next.js 16 + App Router**
- Fully responsive UI with clean gradients and neon accents
- Showcases **Backend + AIML + GenAI Projects**
- Integrated sections for:
  - Skills
  - Education
  - Projects
  - Experience (optional)
  - Contact
- SEO optimized using Next.js metadata
- Fast rendering using Server Components
- Lottie animations for visual enhancement
- Supports Docker deployment
